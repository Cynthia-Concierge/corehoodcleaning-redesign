# Core Hood Cleaning - Redesigned Website

Modern, professional website redesign for Core Hood Cleaning.

## Features

- ✅ Clean, modern design with Core Hood Cleaning brand colors
- ✅ Mobile-responsive layout
- ✅ Professional hero section with NFPA 96 certification badge
- ✅ Complete service listings
- ✅ Customer testimonials section
- ✅ Location-specific pages (San Diego, Chula Vista, La Mesa)
- ✅ Contact form integration ready
- ✅ SEO-optimized content

## Pages

- `index.html` - Homepage with hero, services, testimonials, locations
- `hood-cleaning-san-diego.html` - San Diego location page
- `hood-cleaning-chula-vista.html` - Chula Vista location page  
- `hood-cleaning-la-mesa.html` - La Mesa location page

## Tech Stack

- Pure HTML5/CSS3/JavaScript
- No build process required
- Optimized for performance
- Ready for Vercel deployment

## Deployment

To deploy to Vercel:
1. Import this repo to Vercel
2. Deploy as static site
3. Custom domain: corehoodcleaning.com
